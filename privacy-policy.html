Privacy Policy

Schema Analyzer is a Chrome extension designed to analyze schema markup (JSON-LD and Microdata) available on the currently active webpage.

Data Collection

This extension does not collect, store, transmit, or share any personal or sensitive user data.

  • No personally identifiable information is collected
  • No browsing history is stored
  • No analytics or tracking is performed
  • No data is sent to external servers

How the Extension Works

All schema analysis is performed locally within your browser. The extension only reads structured data present in the HTML of the currently active tab when the user opens the extension popup.

Permissions

Schema Analyzer uses the activeTab permission solely to access the content of the currently active webpage in order to detect and display schema markup at the user’s request.

Third-Party Services

This extension does not use any third-party services, remote code, analytics tools, or external APIs.

Changes to This Policy

This privacy policy may be updated in the future if the extension’s functionality changes. Any updates will be reflected on this page.

If you have any questions about this privacy policy, you can contact the developer via the Chrome Web Store listing.
